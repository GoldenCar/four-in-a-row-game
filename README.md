# Flutter FIAR

Four-in-a-row game in Flutter

## Game modes

- Player vs Player (only local)

- Player vs Cpu

  - Dumb
  
  - Hard
  
  - Hardest
  
- Demo (Cpu Hard vs Cpu Hardest)

## Getting Started

Run the application to play :)

`git clone git@github.com:imtoori/flutter_fiar.git`

`flutter run`
